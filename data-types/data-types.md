# Data-types in golang

## bool
para flag o condicionales
default: false

## uint8, uint16, uint32, uint64, uintptr
entero sin signo (positivo)
default: 0

## int8, int16, int32, int64
entero, controlar tamaño de los enteros (si tiene número)
// --- default: 0

## float32, float64
representar valores numéricos reales
se asocia al sistema. Ej: Windows x32
default: 0

## byte
Alias de uint8
Representar binarios
default: 0

## rune
Alias de uint32
Representar un solo caractér que ocupa más de un byte
default: 0

## complex64, complex128
Para representar una parte real y una imaginaria
default: 0 + 0i

## string
para representar texto
default: ""
